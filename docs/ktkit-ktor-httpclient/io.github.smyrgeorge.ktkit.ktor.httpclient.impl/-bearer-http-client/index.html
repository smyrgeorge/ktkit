BearerHttpClient

class BearerHttpClient(client: HttpClient, baseUrl: String) : AbstractHttpClient(source)

A specialized HTTP client that facilitates interaction with APIs using bearer token authentication. This client provides various HTTP methods such as GET, POST, PATCH, PUT, and DELETE, and supports customization of request headers.

Parameters

client

The base HttpClient used for making HTTP requests.

baseUrl

The base URL for all requests made by this client.
