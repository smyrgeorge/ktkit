XRealNameWebClient

class XRealNameWebClient(client: WebClient, baseUrl: String, val jm: JsonMapper) : AbstractWebClient(source)

XRealNameWebClient is an implementation of AbstractWebClient that provides additional user token handling through the use of the x-real-name header. It simplifies performing various HTTP operations such as GET, POST, PATCH, PUT, and DELETE with pre-set configurations for attaching user tokens.

Parameters

client

The WebClient instance used for making HTTP requests.

baseUrl

The base URL used for constructing request URLs.

jm

The JsonMapper instance responsible for handling JSON serialization and deserialization.
