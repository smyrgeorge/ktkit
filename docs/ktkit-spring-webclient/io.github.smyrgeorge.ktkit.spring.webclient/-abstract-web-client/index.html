AbstractWebClient

abstract class AbstractWebClient(val client: WebClient, val baseUrl: String = "")(source)

An abstract class that provides a foundation for implementing a web client for making HTTP requests.

The AbstractWebClient standardizes the use of a WebClient instance to perform common HTTP operations, such as GET, POST, PATCH, PUT, and DELETE, while offering support for customizing request headers and error handling. This class is designed to streamline client-server communication by encapsulating reusable logic for building requests and processing responses.

Parameters

client

The WebClient instance used for executing HTTP requests.

baseUrl

The base URL of the target API to which requests are made. Default value is an empty string.
