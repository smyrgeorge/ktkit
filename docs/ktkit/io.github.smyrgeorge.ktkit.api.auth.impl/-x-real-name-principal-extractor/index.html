XRealNamePrincipalExtractor

object XRealNamePrincipalExtractor : PrincipalExtractor(source)

Implementation of the PrincipalExtractor interface for extracting a UserToken from an HTTP request's x-real-name header.

This extractor decodes a base64-encoded JSON string from the x-real-name header, deserializes the string into a UserToken object, and returns it as a Result. If the header is missing or the decoding/deserialization fails, the method captures the error and returns it within the Result.
