ErrorSpec

interface ErrorSpec(source)

Represents a specification for errors used within the system.

This interface provides a structured format to define error-related details, including categorization (kind), human-readable message (message), and associated HTTP status (httpStatus). It also offers utility methods to handle errors as throwable exceptions.
