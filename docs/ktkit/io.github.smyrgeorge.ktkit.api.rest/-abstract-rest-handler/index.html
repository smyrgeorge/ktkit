AbstractRestHandler

abstract class AbstractRestHandler(defaultUser: Principal? = null, hasRole: String? = null, hasAnyRole: NonEmptySet<String>? = null, hasAllRoles: NonEmptySet<String>? = null, permissions: HttpContext.() -> Boolean = { true }, principalExtractor: PrincipalExtractor? = null) : Component(source)

Abstract base class for creating REST API endpoints with built-in request handling and security.

This class provides helper methods for defining routes with integrated request processing, user authentication, authorization checks, and response handling.

Parameters

defaultUser

A default principal to use for request handling if no authenticated principal is present.

hasRole

A specific role required for authorization.

hasAnyRole

A set of roles, of which at least one is required for authorization.

hasAllRoles

A set of roles, all of which are required for authorization.

permissions

Additional permission checks to be enforced during request processing.

principalExtractor

Custom principal extractor for user authentication.
