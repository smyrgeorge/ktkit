Var

class Var(kind: HttpContext.Var.Kind, name: String, value: String?)(source)

Represents a variable with a type, name, and an optional value. The variable can be used to retrieve its value in various formats such as String, Long, Int, Float, Double, Boolean, or as an enum value.

Parameters

kind

The type of the variable. It can be one of the values from Kind.

name

The name of the variable.

value

The optional value of the variable.
