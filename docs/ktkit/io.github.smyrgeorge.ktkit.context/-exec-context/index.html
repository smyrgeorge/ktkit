class ExecContext(val reqId: String, val reqTs: Instant = Clock.System.now(), val principal: Principal, val attributes: Map<String, Any> = emptyMap(), val tracing: TracingContext, http: HttpContext? = null, event: EventContext? = null) : Raise<ErrorSpec> , TracingContext, CoroutineContext.Element(source)

Request context containing user information, request metadata, and utilities for accessing request data.

Implements CoroutineContext.Element to allow passing context through coroutines.
